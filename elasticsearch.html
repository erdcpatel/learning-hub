Elasticsearch

Distributed search and analytics engine — full-text search, log analysis, and real-time insights.

Core Concepts

📄 Index
A collection of documents with similar characteristics. Analogous to a database table. Each index has mappings and settings.
📋 Document
A JSON object stored in an index. The basic unit of information. Each document has a unique _id.
🗺️ Mapping
Schema definition for an index — field types (text, keyword, integer, date), analyzers, and indexing options.
🔀 Shard
An index is split into shards for horizontal scaling. Each shard is a Lucene index. Replicas provide fault tolerance.

Query DSL Examples

🔍 Match Query
Full-text search with analyzer: 
GET /articles/_search
{
  "query": {
    "match": {
      "title": "elasticsearch guide"
    }
  }
}
🎯 Bool Query
Combine multiple conditions: 
GET /products/_search
{
  "query": {
    "bool": {
      "must": [
        { "match": { "title": "laptop" } }
      ],
      "filter": [
        { "range": { "price": { "lte": 1000 } } },
        { "term": { "status": "active" } }
      ],
      "must_not": [
        { "term": { "brand": "excluded" } }
      ]
    }
  }
}
📏 Range Query
Filter by numeric or date range: 
GET /logs/_search
{
  "query": {
    "range": {
      "@timestamp": {
        "gte": "now-1h",
        "lte": "now"
      }
    }
  }
}
🔗 Nested Query
Query nested objects: 
GET /orders/_search
{
  "query": {
    "nested": {
      "path": "items",
      "query": {
        "bool": {
          "must": [
            { "match": { "items.name": "widget" } },
            { "range": { "items.qty": { "gte": 5 } } }
          ]
        }
      }
    }
  }
}

Aggregations

GET /orders/_search
{
  "size": 0,
  "aggs": {
    "sales_per_month": {
      "date_histogram": {
        "field": "order_date",
        "calendar_interval": "month"
      },
      "aggs": {
        "total_revenue": {
          "sum": { "field": "amount" }
        },
        "avg_order_value": {
          "avg": { "field": "amount" }
        }
      }
    },
    "top_categories": {
      "terms": {
        "field": "category.keyword",
        "size": 10
      }
    }
  }
}

Mapping & Analyzers

PUT /articles
{
  "settings": {
    "number_of_shards": 3,
    "number_of_replicas": 1,
    "analysis": {
      "analyzer": {
        "custom_analyzer": {
          "type": "custom",
          "tokenizer": "standard",
          "filter": ["lowercase", "stop", "snowball"]
        }
      }
    }
  },
  "mappings": {
    "properties": {
      "title":     { "type": "text", "analyzer": "custom_analyzer" },
      "slug":      { "type": "keyword" },
      "body":      { "type": "text" },
      "author":    { "type": "keyword" },
      "published": { "type": "date" },
      "views":     { "type": "integer" },
      "tags":      { "type": "keyword" },
      "location":  { "type": "geo_point" }
    }
  }
}
