Quick Start

Get up and running in seconds

🚀 Using This Hub

Locally: Just clone the repo and open any .html file in your browser. No server needed.


Online: Access via GitHub Pages at your repository URL.


Add a prompt: Edit prompts.html and add an object to the promptsData array.


Add a topic: Copy PAGE_TEMPLATE.html, rename, fill content, and update the nav links in js/common.js.
